For This Task

This brief aims to be a basic introduction to the web, how the web works and how text and image work online. We will also introduce you to your own web space that you will be able to use throughout your Visual Communication degree. Sometimes you might use this for course work. But you are also free to use this as a play space!

AThis brief is inspired by Vanessa Berry's Mirror Sydney book, an alternative "map" of Sydney that considers odd, forgotten and abandoned places of the city.

You will design a simple web page that records your encounter with a place in Sydney. You are asked to make an illustration of that place or structure and to write a brief description of your encounter, memories or response to it. We have selected some of the themes from Vanessa Berry's book that you should use as a guide as to what types of places you record. These include:

  • Sydney Bestiary
  • Mystery Structures
  • Memorial Sites
  • Murals and Ghost Signs
  • Suburban Time

We will show some images of these categories in the lecture, so see the lecture slides for some guidances as to what you might consider for these categories.

We want to you to consider the typographic detailing (font, colour, size, spacing), the visual hierarchy and the combination of text and image on your web page. Your illustration of your place can be done in any form you feel comfortable, it can be hand-drawn, a collage, a tracing...

Individually you will be creating a web site on a single place you have noticed or that you remember. But as a group (i.e. the whole of Text and Image II) you will be making your own alternative map of Sydney. As your websites will be linked with each other, meaning that a person on the web can navigate between your pages. We will help you to link between your websites with each other in class.

Requirements / How to do This Task

We will be giving you a base file to work on that includes some basic text. Stick to the base code, and using what we teach you in class, we want you to focus on the design of the details of the page (i.e. illustration and typography). You can download the class examples and the base file to work from here Intro to Web Resources. The base file is session.1 > 1.4-page-with-copy-text

Use CSS (Cascading Style Sheets) to alter the typography of the layout. You should seek to create a visual hierarchy in the typography (i.e. using the typographic detailing such as font type, style, colour and white space)

You can use the generic web safe fonts (i.e. ariel, times, verdana). But you should also aim to use web fonts. As they are free and accessible we will be using Google Web Fonts in class and recommend you stick to them for your assignment (for now!)

Consider how your illustration works with your choices as regards to your typography. How do the text and image stylistically work together in terms of their visual qualities?

Working on Your Design

You can work directly with the code in order to achieve your design. Sometimes however it can be quicker to sketch out on paper what you are imagining it will look like. You can use thumbnail sketches to quickly detail the layout and visual hierarchy of your page.
